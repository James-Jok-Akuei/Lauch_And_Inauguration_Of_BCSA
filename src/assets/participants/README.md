# Participant gallery images

Each folder here is one university or association (named by its `slug` in
`src/content.js`). To fill a group's slideshow, just DROP image files into its
folder — no code changes needed. They appear in the gallery in filename order.

Supported: .jpg .jpeg .png .webp .gif .avif
Tip: name files 01.jpg, 02.jpg, ... to control the order.

Folders:
  Universities: mku, uok, alu, ur, ulk, upi
  Associations: south-sudanese, twic-mayardit, twic-east, madi,
                eastern-equatoria, wau, fashoda, bari, ram, azande, nuer,
                renk, pojulu, aweil
